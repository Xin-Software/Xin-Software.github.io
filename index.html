Our Products

Xin List

Xin List is a simple, easy-to-use application that helps you manage your lists and stay organized.

Yep, it's essentially a TODO list... we've all made them. But this one is great! Xin List lets you create list items and assign them into categories, reorganise and filter them and check them as complete.

Xin Clip

Do you keep a notepad with an innumerable count of tabs containing miscellaneous passwords, code snippets, ...?

Well I did, and it was a mess, so we made Xin Clip.

Xin Clip stores blocks of text in a user-friendly collection of "clips". You can quickly copy and paste content to clips, giving them sensible names and descriptions for easy accessibility. Xin Clip stores all clip data securely in an encrypted file, with an option to obscure sensitive information on the UI so all of those SQL passwords you have lurking around are better secured
